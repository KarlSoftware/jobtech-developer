Historical Job Postings


Description

This dataset consists of 4.2M job postings published on Platsbanken from the year 2006 up to and including 2016. Texts have been anonymized with sentences including names, telephone numbers and email adresses removed. Each listing contains metadata on location, dates, employer name, job type and job details.

For a visual overview of this dataset, please see the dashboard example.

API with basic statistics

We provide a microservice for basic statistics and queries on the dataset. A notebook with examples can be found here (under construction).

Data format

Each line in a file contains a job listing in a JSON format (JSON Lines):

...
{"ANTAL_AKT_PLATSER": "3", "PLATSBESKRIVNING": "...", ... }
{"ANTAL_AKT_PLATSER": "1", "PLATSBESKRIVNING": "...", ... }
{"ANTAL_AKT_PLATSER": "1", "PLATSBESKRIVNING": "...", ... }
...

A listing has the following fields:

FieldDescription
ANTAL_AKT_PLATSER
PLATSBESKRIVNINGAd text
ADRESSLANDCountry (Sweden if not specified)
AG_NAMNEmployer name
ARBETSDRIFT
ARBETSTID
BESKR_ARBETSDRIFT
FORSTA_PUBLICERINGSDATUMPublication date
KOMMUN_KODMunicipality code
PLATSNUMMER
PLATSRUBRIK
POSTORT
SISTA_ANSOK_PUBLDATUM
TILLTRADE
VARAKTIGHET
YRKE_IDSSYK occupation id

All fields are strings and may be nullable, i.e. no data may be given for a field.
